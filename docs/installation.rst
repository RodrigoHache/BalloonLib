Installation
============

Requirements
------------

- Python >= 3.9
- PyTorch >= 2.0
- CUDA-capable GPU (recommended)

Install from source
-------------------

.. code-block:: bash

   git clone https://github.com/errehache/BalloonLib.git
   cd BalloonLib
   pip install -e .

For development (includes testing and linting tools):

.. code-block:: bash

   pip install -e ".[dev]"

For documentation building:

.. code-block:: bash

   pip install -e ".[docs]"
