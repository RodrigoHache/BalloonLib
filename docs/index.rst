BalloonLib
==========

**HRF reconstruction via the Balloon Model using Physics-Informed Neural Networks.**

BalloonLib provides a PyTorch-based framework for reconstructing hemodynamic
response functions (HRFs) from fMRI BOLD signals using the Balloon model
(Buxton 1998; Stephan 2007) embedded in a physics-informed neural network
(PINN).
